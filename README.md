## 주요기능

1. 유튜브 API 활용
2. 인기 동영상 목록 보기
3. 키워드로 동영상 검색
4. Tailwind CSS로 구현한 반응형 웹 페이지
5. 라이트 모드 / 다크 모드 테마
6. 로컬스토리지에 테마 저장하여 기억
